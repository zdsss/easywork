## 公共约定

### Token 处理
- 登录成功后存入 `localStorage`（key: `token`）
- Axios 请求拦截器自动加入 `Authorization: Bearer <token>`
- 401 响应时自动跳转登录页

### API 响应格式
```json
{ "code": 200, "message": "Success", "data": {...}, "timestamp": 1234567890 }
```
- `code === 200` 时返回 `data`
- 其他 code 时弹出 `message` 错误提示
